The Daily Quote & Inspiration Application is intended to promote a healthier mindset through daily inspirational quotes that can boost productivity and improve mood. With this application, users have the ability to use generate a random quote and use the voice command to say how they feel (happpy, sad, neutral). I hope that my application will enhance the lives of many in a positive and healthy way.

For this project the following applications were used: html, javascript, css, fetch api, typed.js, annyang.js
